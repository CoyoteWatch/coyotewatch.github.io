Создание Telegram-ботов для сбора данных

О проекте

Проект Coyotewatch позволяет создавать Telegram-ботов (логгеров), которые собирают информацию о пользователях, взаимодействующих с ботом через веб-приложения. Система предоставляет инструменты для анализа данных и настройки ботов под ваши требования.

Как это работает?

Пользователь взаимодействует с ботом, который имитирует полезный сервис. В процессе использования бот запрашивает подтверждение номера телефона или другие данные. При согласии пользователя его контактные данные становятся доступны владельцу бота.

Важно: Подобные методы могут нарушать правила платформы и законодательство. Используйте их исключительно в ознакомительных целях.

Создание бота

  1. Откройте бота @CoyoteWatch_bot и отправьте токен вашего бота.
  2. Следуйте пошаговым инструкциям.
  3. Настройте бота согласно вашим требованиям.
  4. После прохождения проверки жертвой собирается техническая информация:
    • Идентификатор аккаунта
    • Имя пользователя
    • IP-адрес
    • Данные об устройстве
    • User-Agent браузера

Для пользователей

После запуска бота и принятия пользовательского соглашения отображается заранее подготовленный текст. Появляется кнопка для перехода в мини-приложение с проверкой CAPTCHA. Повторные попытки вызывают ложное срабатывание защиты от спама.
